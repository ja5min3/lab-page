🖨️ Bambu Lab H2D（主力機）

基本資訊

  • 切片軟體：Bambu Studio
  • 材料：PLA、PETG、ABS／ASA、TPU
  • 送料：AMS 自動換料＋外掛料架都有（常用料進 AMS，特殊料走外掛）

操作步驟

  1. 開機與檢查：確認平台乾淨、料量足夠
  2. 上料／換料：常用料放 AMS；特殊料（如 TPU）掛外掛料架
  3. 切片：Bambu Studio 選對材料與板面設定
  4. 傳檔：依規定用隨身碟（機台盡量不聯網）
  5. 列印中注意：頭幾層留意第一層有沒有黏好
  6. 取件與收尾，並完成使用後表單登記

第一層黏不住／翹曲怎麼救

  1. 拿中性洗碗精洗打印板——紋理板可用菜瓜布刷；光面板只能用海綿，菜瓜布會刮花！
  2. 上膠水／平台膠
  3. 改切片設定（加 brim、調溫度）

耗材存放

  • 開封的線材：防潮箱＋AMS 裡
  • 未開封：封好的紙箱
  • 常用小量：好拿的箱子

🖨️ Creality Ender 3 S1（備用機）

需受訓才能用——這台要手動調平，沒有 H2D 的自動校正，調不好第一層就失敗。想學找 3D 列印區管理人。

調平（受訓重點）

  1. 自動調平：開機 → 螢幕選「Level」→ 自動歸位並做 16 點校正
  2. 設 Z-offset：Prepare → Z-offset，調到噴嘴與平台距離約一張 A4 紙厚（0.08–0.1mm），按旋鈕確認
  3. 輔助調平（平台傾斜 >2mm、自動調平失敗時）：Control → Reset configuration → Auto Home → Move Z 歸零 → 設 Z-offset → Disable stepper → 轉熱床底下四角旋鈕，讓四角都是一張 A4 紙的距離
  4. 判斷：太遠→黏不住；太近→擠不出、刮平台；剛好→線材均勻擠出平貼

上料

  1. 線材尾端剪 45° 斜角
  2. 穿過斷料偵測器的孔 → 按住擠出機把手 → 沿孔插到噴嘴位置
  3. 預熱噴嘴，到溫後看到料流出＝完成

換料

  • 沒在列印：噴嘴加熱到 185°C 以上 → 按住把手快速抽出（慢慢拉會卡管）→ 照上料步驟換新料
  • 列印中：先暫停 → 停穩後抽出 → 穿新料 → 先擠掉殘料再續印

列印

  1. 切片（Creality Slicer／Cura）存到記憶卡
  2. ⚠️ 檔名只能英文或數字——中文檔名螢幕顯示不出來
  3. 插卡 → 旋鈕選檔 → 開印

安全注意（原廠警告）

  • 開機狀態不要用手移動噴嘴或平台（會觸發保護斷電）
  • 噴嘴和平台高溫，列印中勿碰；清噴嘴一定用工具
  • 清潔前先關電源；平台可用玻璃清潔劑或 IPA 擦

🖨️ San Draw S050 矽膠列印機

需受訓才能用——矽膠列印流程跟 FDM 完全不同，弄錯會浪費貴料或堵管。想學找 3D 列印區管理人。

基本資訊

  • 技術：FAM（Fluid Additive Manufacturing，三遞專利）
  • 軟體：三遞原廠軟體（FAMufacture／Cura）
  • 材料：SIL28（較硬）、SIL18（較軟），依研究需求選硬度
  • 固化：自然固化即可，不用烤箱

⚠️ 最容易出事的環節

  1. 校正——沒校好直接影響成敗，列印前務必確實
  2. 擠料異常要警覺——擠料不順可能是該換噴頭的訊號，硬印會失敗
  3. 裝料／換料——步驟特殊，沒學過不要自己來
  4. 噴頭堵塞——矽膠在管路裡固化就麻煩了

🎓 3D 列印使用考核制度（草案）

草案尚未定案，以管理人公告為準。目標：機器貴、耗材貴，確保每個人具備基本能力再上機。

Lv.1 基礎（H2D）

  • 問答：登記規則（前紙本後表單）、傳檔規定（隨身碟）、打印板清潔（紋理板 vs 光面板）
  • 實作：在管理人陪同下完成一次完整列印（切片 → 傳檔 → 上料 → 列印 → 取件 → 登記）

Lv.2 進階（＋Ender）

  • 實作：手動調平到管理人認可
  • 問答：調平失敗會怎樣、怎麼判斷第一層有沒有黏好

Lv.3 矽膠（＋S050）

  • 受訓：管理人帶一次完整流程（裝料、校正、列印、清潔）
  • 問答：擠料不順代表什麼、哪些環節最容易出事
  • 實作：獨立完成一件並通過檢查

🛡️ 每學期初自主檢查（8 項）

  1. 實習場所安全衛生認證制度查核表——線上（實驗室安全巡檢管理系統，校內網）或紙本二擇一
  2. 自動檢查紀錄——列管檢查、定期檢查（排氣裝置、滅火器等）、作業檢點
  3. 安全衛生告知單——每次實驗課講解安全規定並簽名，紀錄保存 3 年
  4. 消防設備自動檢查——消防栓箱、緊急照明、出口標示燈
  5. 防護器具——依實驗性質建清冊，實驗中確實配戴
  6. 化學藥品管理——清單每學年更新，依性質分類儲放
  7. 實驗室環境清潔
  8. 有機溶劑＋特化作業檢點表——紙本填寫、留室備查（本實驗室不用鋼瓶表）
